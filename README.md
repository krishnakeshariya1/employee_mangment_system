🚀 Employee Management System

A simple Employee Task Management System built with React and Vite.
The system allows an Admin to create and assign tasks to employees, while Employees can log in and view their assigned tasks.

The application also includes basic authentication and role-based access control using LocalStorage and React Context API.

✨ Features

✔ Admin and Employee Login
✔ Role-based Dashboard (Admin / Employee)
✔ Admin can create tasks
✔ Admin can assign tasks to employees
✔ Employees can view their assigned tasks
✔ Authentication using LocalStorage
✔ Global state management using Context API
✔ Persistent data using LocalStorage
